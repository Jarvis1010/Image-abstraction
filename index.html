How to Use this API

To use this api, add '/api/imagesearch/' to the address above and the search term to get the 20 most recent image abstractions as an array of JSON objects. Optionally you can add '?offset=' to add a number to offset by.

Example: /api/imagesearch/mario brothers Example: /api/imagesearch/mario brothers?offset=20

In addition, if you add '/api/latest/imagesearch/' to the above string, you get the most recent 20 search queries returned.
